アプリ概要

Fonts Social 写真文字入れ-日本語English APP は、画像やキャンバスに 3D の文字を挿入できるシンプルなアプリです。

主な特長

  • 日本語、英語の 3D フォントが利用できます。
  • 縦書きに対応しています。
  • 文字のサイズを自由に変更できます。
  • 文字の色を自由に変更できます。
  • 文字の背景色を自由に変更できます。
  • 文字を立体的に表現（3D）できます。
  • 文字の間隔（カーニング）を自由に変更できます。
  • 行の間隔を自由に変更できます。
  • 文字の傾きを自由に変更できます。

3D【world】

3D【world】では猫が表示されます。実機でこの流れを試してみてください。

  1. 3D → Hide UI
  2. 猫だけ見えるか確認
  3. 画面タップで UI が戻るか確認
  4. Shoot
  5. 写真に保存 / 共有 の選択が出るか確認
ライブカメラを使うときは、3D → Camera On のあとに外背景が live で見えるか確認し、 Shoot を押して share sheet まで進めるかを見てください。

Vector

Vector では、日常を整えるリマインダーアプリ的な使い方もできます。

  • タイマー、カレンダー予定、リマインダーを簡単にセットできます。
  • すべて繰り返し利用に対応しています。
  • タイマーは画面全体に通知されます。
  • 設定できる期間に上限はありません。

Vector ツールガイド

  • ボード は、アートボードを追加・複製・削除するツールです。
  • ブレンド は、1つ目をタップ → 2つ目をタップ で試せます。
  • パス は、ペンや曲線で作る自由な線や形のことです。
  • ワープ は、変形したいオブジェクトをそのまま指でドラッグして使います。
  • スポイト は、項目をタップして吸い取り、必要なら適用します。
  • 形成 は、1個目の閉じた図形をタップ → 2個目の閉じた図形をタップ で即合体します。
  • ハサミ は、線の近くを 1 回タップするとそのまま切れる挙動です。
  • ペン は、ブラシツール内のペンでも指入力に対応しています。
パスは ペン / 曲線 で作る自由形状です。長方形や円は見た目が線でも専用図形なので、 今のワープでは本格的な変形対象ではありません。ワープは ペン → 1本描く → 確定 → ワープ → 線をドラッグ が一番分かりやすいです。
ボードの使い方: ボード を選ぶ → 上の小さいバーで操作 → ＋ で追加 → 重なった四角で複製 → ゴミ箱で削除。1枚しかないときは削除できません。

Photo

Photo は、高機能ツールを揃えた画像編集アプリです。

  • 明るさ、彩度を調整できます。
  • 写真、画像、デザインをさまざまな方法でカスタマイズできます。
  • 複数のファイル形式に対応しています。
  • Photo → スタジオ → ルック → Noir を押すと、キャンバスと全体プレビューの両方で変化が見えます。

Photo の詳細ツール

今の実機 iPhone で見える主な項目は次のとおりです。

  • 基本補正 は 補正 タブにあります。
  • 白黒 は Monochrome と B/W mixer として見えます。
  • 粒子 は Grain として見えます。
  • ノイズ除去 は見えます。
  • カラーグレーディング は見えます。
  • カラーミキサー は見えます。

条件付きで見えるもの:

  • 文字間調節 はテキストレイヤー選択時に 字間 / 行間 として見えます。
  • グラデーションマップ は調整レイヤー選択時に見えます。
  • RAWディテール は Studio > Desktop Panels > Camera Raw 側にあります。
  • フィルタ は独立パネルではなく、Blur / Motion Blur / Sharpen / Glow などが基本補正側へ分散しています。

スーパー解像度

今回の版では Photo → 補正 → 詳細ツール → スーパー解像度 → 2X が使えます。

  • 高画質を狙いやすく、細部補完や質感再現に向いています。
  • ただし処理は重く、メモリや発熱の影響を受けやすいです。
  • 高精細 は 51、創造性 は 50、標準 は 54 で止まります。
  • 創造性 はディテール補完が 50 まで、高精細 は 51 までです。

"Board"... Board is a tool for adding, duplicating, and deleting artboards. Use it like this.
1. Select Board at the top.
2. Operate with the small bar that appears at the top.
3. Tap + to add a new artboard.
4. Tap the overlapping squares to duplicate the current artboard.
5. Tap the trash can to delete the current artboard.
However, you cannot delete it when there is only one artboard.

"Blend"... Try Blend -> tap the first object -> tap the second object.

"What is a path?" A path is a free line or shape created with Pen or Curve.
In this app:
・ Rectangle is a rectangle
・ Circle is a circle
・ Line is a straight line
・ Path is a freeform shape created with Pen / Curve
For example:
・ a line created by placing points with Pen
・ a curve created with Curve
These are paths.
On the other hand, rectangles and circles are dedicated shapes internally, so Warp does not fully work on them.
To try Warp on an actual device, the easiest way is:
select Pen -> draw one line -> confirm -> Warp -> drag that line.
In short:
・ Path = freely drawn line or shape
・ Rectangle / Circle = dedicated shapes, not paths

"Warp"... Use it by dragging the object you want to transform directly with your finger.
How to use:
・ Prepare one path
・ Select Warp at the top
・ Drag directly on the line you want to deform
・ Release your finger to confirm the shape
Notes:
・ Nothing happens if you drag empty space
・ Grab the object itself, not just near the line
・ Current Warp behaves more like local path drag deformation than a general warp tool

"Eyedropper"... Eyedropper -> tap an item to pick it -> apply if needed.

"Shape Builder"... It now remembers the first shape.
・ Tap the first closed shape
・ Tap the second closed shape to merge immediately
If the second one is hard to tap because of overlap, tap the first one again and it will merge with the overlapping target automatically.
(Example: overlap two rectangles slightly -> Shape Builder -> first tap -> second tap)

"Scissors"... In Vector, try tapping once near a line with Scissors. It is tuned so one tap near the line cuts it immediately.

"Pen"... The case was the Pen inside the Brush tool on the top row. On actual devices, PKCanvasView had been Apple Pencil only, so you could not draw with your finger.
It was fixed by changing drawingPolicy to anyInput so you can draw directly with your finger on iPhone.

Black & White apparaît comme Monochrome et B/W mixer.

개요

Fonts Social 写真文字入れ-日本語English는 이미지와 캔버스에 3D 텍스트를 배치할 수 있는 간단한 앱입니다.

  • 일본어와 영어 3D 폰트
  • 세로쓰기 지원
  • 크기, 색상, 배경색, 자간, 줄간격, 기울기를 자유롭게 조절 가능
  • 창의적인 레이아웃을 위한 3D 텍스트 스타일

3D / world3

  • 3D -> Hide UI 로 고양이만 보이게 테스트해 보세요.
  • 화면을 탭하면 UI 가 다시 나타나는지 확인하세요.
  • Shoot 후 저장 / 공유 선택이 뜨는지 확인하세요.
  • 라이브 카메라 모드에서는 Camera On 후 바깥 배경이 live 로 보이는지 확인하세요.

Vector

  • 타이머, 캘린더 일정, 미리 알림을 만들고 반복해서 사용할 수 있습니다.
  • Board 는 아트보드를 추가, 복제, 삭제합니다.
  • Blend 는 선택한 두 개의 오브젝트를 합칩니다.
  • Path 는 Pen 또는 Curve 로 만든 자유형 선/도형입니다.
  • Warp 는 오브젝트를 직접 드래그해서 사용합니다.
  • 스포이트는 색을 추출하고 적용합니다.
  • Shape Builder 는 닫힌 도형을 합칩니다.
  • Scissors 는 선 근처를 탭해서 자릅니다.
  • Brush 도구 안의 Pen 은 iPhone 에서 손가락 입력을 지원합니다.

Photo

  • 밝기, 채도 등 다양한 이미지 조정
  • 여러 파일 형식 지원
  • Noir 같은 룩 프리셋
  • 흑백, 그레인, 노이즈 제거, 컬러 그레이딩, 컬러 믹서 등 세부 도구
  • 슈퍼 해상도는 현재 2X 만 지원합니다.

<Vector에서는 일상을 정리하는 리마인더 앱처럼 사용할 수 있습니다.>
・ 타이머, 캘린더 일정, 리마인더를 쉽게 설정할 수 있습니다. 모두 반복 사용이 가능합니다.
・ 타이머는 전체 화면 알림으로 표시됩니다.
・ 설정 가능한 기간에는 상한이 없습니다.

"Board"... Board 는 아트보드를 추가, 복제, 삭제하는 도구입니다. 사용 방법은 다음과 같습니다.
1. 위쪽의 Board 를 선택합니다.
2. 위에 나타나는 작은 바에서 조작합니다.
3. + 로 새 아트보드를 추가합니다.
4. 겹쳐진 사각형 버튼으로 현재 아트보드를 복제합니다.
5. 휴지통 버튼으로 현재 아트보드를 삭제합니다.
단, 아트보드가 1개뿐일 때는 삭제할 수 없습니다.

"Blend"... Blend -> 첫 번째 오브젝트 탭 -> 두 번째 오브젝트 탭 순서로 시도해 보세요.

"Path란?" Path 는 Pen 이나 Curve 로 만드는 자유로운 선과 도형입니다.
이 앱에서는 대략 다음과 같이 나뉩니다.
・ Rectangle 은 사각형
・ Circle 은 원
・ Line 은 직선
・ Path 는 Pen / Curve 로 만드는 자유형
예를 들면:
・ Pen 으로 점을 찍어 만든 선
・ Curve 로 만든 곡선
이것들이 path 입니다.
반면 Rectangle 과 Circle 은 내부적으로 전용 도형이기 때문에 현재 Warp 가 본격적으로 적용되지는 않습니다.
실기기에서 Warp 를 시험해 보려면:
Pen 선택 -> 선 하나 그리기 -> 확정 -> Warp -> 그 선을 드래그
가 가장 이해하기 쉽습니다.
정리하면:
・ Path = 자유롭게 그린 선이나 도형
・ Rectangle / Circle = path 가 아닌 전용 도형

"Warp"... 변형하고 싶은 오브젝트를 손가락으로 직접 드래그해서 사용합니다.
사용 방법:
・ Path 를 하나 준비합니다
・ 위쪽의 Warp 를 선택합니다
・ 변형하고 싶은 선 위를 직접 드래그합니다
・ 손을 떼면 그 모양으로 확정됩니다
보충:
・ 아무 것도 없는 곳을 드래그해도 아무 일도 일어나지 않습니다
・ 선 근처가 아니라 오브젝트 자체를 잡고 드래그합니다
・ 현재 Warp 는 일반 변형보다 국소적인 path drag 변형에 가깝습니다

"Eyedropper"... Eyedropper -> 항목을 탭해서 색을 흡수 -> 필요하면 적용.

"Shape Builder"... 지금은 첫 번째 도형을 기억하도록 되어 있습니다.
・ 첫 번째 닫힌 도형을 탭합니다
・ 두 번째 닫힌 도형을 탭하면 즉시 합쳐집니다
겹쳐 있어서 두 번째가 탭하기 어렵다면 첫 번째를 한 번 더 탭하면 겹친 상대와 자동으로 합쳐집니다.
(예: 사각형 두 개를 조금 겹치기 -> Shape Builder -> 첫 번째 탭 -> 두 번째 탭)

"Scissors"... Vector 에서 Scissors 로 선 근처를 한 번 탭해 보세요. 현재는 선 근처를 한 번 탭하면 바로 잘리도록 조정되어 있습니다.

"Pen"... 문제는 상단 브러시 도구 안의 Pen 이었습니다. 실제 기기에서는 PKCanvasView 가 Apple Pencil 전용이어서 손가락으로는 그릴 수 없었습니다.
이를 drawingPolicy 를 anyInput 으로 바꿔 iPhone 에서도 손가락으로 바로 그릴 수 있게 수정했습니다.

<Photo 는 고급 도구를 갖춘 이미지 편집 앱입니다.>
・ 밝기와 채도를 조정할 수 있습니다.
・ Photo 를 사용하면 사진, 이미지, 디자인을 다양한 방법으로 커스터마이즈할 수 있습니다.
・ 여러 파일 형식을 지원합니다.
・ Photo -> Studio -> Look -> Noir 를 눌러 보세요. 캔버스와 전체 프리뷰 양쪽에서 변화가 보일 것입니다.
・ 이번 버전에서 Photo 의 super resolution 은 2X 만 지원합니다.
    * 더 높은 화질을 노릴 수 있습니다
    * 세부 보완과 질감 재현은 CI 보다 유리합니다
    * 다만 무겁습니다
    * 학습 모델 생성, 양자화, 변환, 기기별 테스트가 필요합니다
    * 앱 크기, 메모리, 발열, 처리 시간을 관리해야 합니다
(Photo -> Adjust -> Detailed Tools -> Super Resolution -> 2X)
Super Resolution 의 Creativity 는 실제 기기에서 50 까지만 올라가도록 되어 있습니다.
변경 사항:
* Creativity 의 상한을 50 으로 추가
* 모델 전환 시 자동으로 그 상한으로 보정
* UI 슬라이더도 Creativity 선택 시 0...50 사용
High Detail 은 51, Creativity 는 50, Standard 는 54 에서 멈춥니다.
